Intro-to-latex

This contains files and documentation for the intro to LaTeX workshop put on by the physics society at UOIT.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the basics of LaTeX workshop held by the Physics and Health Physics Societies at UOIT!

We will be using Texmaker as our LaTeX editor. Feel free to use your own software if you prefer, but keep in mind that all of our examples and instructions will be based on Texmaker. If you do choose to use Texmaker, please download it ahead of time so that we can get started right away.

We will be covering the following basic topics:

  • Adding figures

  • Making lists

  • Math mode

  • Creating tables

  • Title pages

  • Making a table of contents

The workshop will continue to build on one document but standalone examples can be found in the Examples folder. To get these files you can download the zipfile (or tarball if you are running Linux) that is linked on the top of this page. Alternatively you can clone the repository by running the command 

$ git clone https://github.com/uoit-research-workshops/intro-to-latex
